# Security Policy / セキュリティポリシー

## Supported Versions / サポートバージョン

現在サポートしているバージョン：

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability / 脆弱性の報告

セキュリティ上の問題を発見した場合：

1. 公開Issueには投稿しないでください
2. ask@k.lne.st に詳細を送信
3. 修正後にセキュリティアドバイザリを公開

報告に含めるべき情報：

- 脆弱性の説明
- 再現手順
- 影響範囲
- 可能であれば修正案
